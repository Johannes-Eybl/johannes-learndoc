# Cloud Computing erklärt
Cloud Computing basiert auf dem Prinzip, dass einzelne Arbeitsbereiche nicht mehr auf der eigenen Festplatte, sondern über das Internet oder ein lokales Netzwerk zur Verfügung stehen.

Dabei können einzelne Programme, Softwarepakete, Speicherplatz und oder Rechenleistung aus einer externen virtuellen Umgebung kommen.

Besonders nützlich ist Cloud Computing, wenn man bestimmte Ressourcen nur für eine gewisse Zeit braucht. Es erlaubt, seine Ressourcen flexibel anzupassen.

# Vorteile von Cloud Computing
- Verfügbarkeit
- Skalierbarkeit
	- Vertikales Skalieren -> Mehr RAM oder CPUs
	- Horizontales Skalieren -> Mehr virtuelle Maschinen
- Elastizität
- Flexibilität
- Geografische Verteilung
- Notfallwiederherstellung
